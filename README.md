# Business Insight — Mobile App

A full-stack Business Insights mobile app with interactive KPI dashboards and data filtering. Built as a full-stack assignment.

## Features

- KPI dashboard with charts and trend indicators
- Filter data by date range, category, and region
- REST API backend with MongoDB storage
- Cross-platform mobile UI (iOS + Android)

## Project Structure

```
client/          # React Native / Expo app
server/          # Node.js + Express + MongoDB API
```

## Run

```bash
# Backend
cd server && npm install && npm start

# Mobile
cd client && npm install && npx expo start
```

## Tech

React Native · Expo · Node.js · Express · MongoDB · Recharts
